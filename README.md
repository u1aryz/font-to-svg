# font-to-svg
Generates SVG file from text and font files using Node.js.

Output sample:

<img width="400px" src="art/sample1.svg" alt="sample">
<img width="400px" src="art/sample2.svg" alt="sample">

## Getting started
```bash
pnpm i
```

To Run:
```bash
pnpm start
```
